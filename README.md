# Vendora - Vendor Management System

## 🚀 Overview
A full-stack application to manage vendors, approvals, and product workflows.

## 🛠 Tech Stack
- Java (Servlet)
- JSP
- MySQL

## ✨ Features
- Role-based access (Admin/User)
- Vendor approval workflow
- Validations

## ⚙️ Setup
1. Clone repo
2. Run backend
3. Run frontend
