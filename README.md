# Vendora - Vendor Management System

## 🚀 Overview
A full-stack application to manage vendors, approvals, and product workflows.

## 🛠 Tech Stack
- Java (Servlet)
- JSP
- MySQL

## ✨ Features

- Role-based authentication (Admin/User)
- Vendor approval workflow (Pending → Approved/Rejected)
- Vendor CRUD operations (Add, Update, Delete)
- Input validation and error handling
- Search and filtering functionality

## 🏗️ Project Architecture

- MVC Pattern (Model-View-Controller)
- Servlet handles request processing
- JSP used for frontend views
- MySQL used for data persistence

## 🚀 Future Enhancements

- Integration with Spring Boot for REST APIs
- Improved UI/UX with modern frontend frameworks
- Email notifications for vendor approval
- Pagination and performance optimization

## ⚙️ Setup Instructions

1. Clone the repository  
2. Import project into Eclipse IDE  
3. Configure MySQL database (update DB credentials)  
4. Deploy on Apache Tomcat Server  
5. Run the application
